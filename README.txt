A simpleminded synthesizer implemented in ActionScript/Flash 10

(c) 2009 Joost Diepenmaat, joost@zeekat.nl
Demo page: http://joost.zeekat.nl/bassline.html

To compile this, you need the free Flex 3 SDK from
http://www.adobe.com/products/flex/flexdownloads/

Then make sure the paths in build.sh are correct and run it.

There's a little more info at 
http://joost.zeekat.nl/2009/03/19/a-simple-synthesizer-in-717-kb-of-flash/
